API Endpoints

Property Endpoints

GET /api/Property - Get all properties
GET /api/Property/{id} - Get property by ID
GET /api/Property/category/{category} - Filter by category
GET /api/Property/propertytype/{type} - Filter by type
POST /api/Property - Create new property
DELETE /api/Property/{id} - Delete property

User Endpoints

GET /api/User - Get all users
POST /api/User/login - User authentication

Agent Endpoints

GET /api/Agent - Get all agents
GET /api/Agent/{id} - Get agent by ID
POST /api/Agent - Create new agent
PUT /api/Agent/{id} - Update agent
DELETE /api/Agent/{id} - Delete agent

Appointment Endpoints

GET /api/Appointment - Get all appointments
POST /api/Appointment - Book appointment
PUT /api/Appointment/status/{id} - Update status

Database Schema

Core Tables & Relationships

Users Properties PropertyImages
Agents Appointments Feedback
Properties Table Fields
  • PropertyId (PK)
  • Title, Description
  • PropertyTypeId (FK)
  • Category (Buy/Rent/Sell)
  • Price, Location, Area
  • Bedrooms, Bathrooms, Amenities
  • Status (Available/Sold/Pending)
PropertyImages Table Fields
  • ImageId (PK)
  • PropertyId (FK)
  • ImageName, ImagePath
  • IsPrimary (Boolean)
  • DisplayOrder (Int)

How to Run

1
Create Database
sqlcmd -S PRIT-LEGION -E -i RealEstateDB_Clean.sql
2
Start API
dotnet run --project RealEstate_Api
3
Start Apps
dotnet run --project RealEstate_Admin
dotnet run --project RealEstate_Frontend
